# [LeetCode 581] 两种常规思路，排序 + 双指针 && 单调栈

暴力法和一些只能用于特定题目的解法可能不太具有借鉴意义，所以这里主要是介绍解决这个题目的两种常规思路。

# 排序 + 双指针

从题意中看，如果对目标子数组进行升序排序，那么整个数组都会变为升序排序，那么最直观的想法就是，现在的数组和升序排序后的数组有什么不同？我只需要找到排序前后数组的不同，然后对这个不同点进行升序排序，就能让整个数组有序。所以，我们的目的就是找到这个不同点，然后输出它的长度：

function findUnsortedSubarray(nums: number[]): number {
  const tmp = [...nums].sort((a, b) => a - b);
  let b = 0, e = nums.length - 1;
  while (b < e && nums[b] == tmp[b]) ++b;
  while (b < e && nums[e] == tmp[e]) --e;
  return e - b + Number(b !== e);
}

也许这个不算是标准的双指针，但是思路确实是双指针的思路，从两端向中间靠近。

最后那一行的Number(b !== e)是啥意思呢？设想一下，如果b和e两个指针重合，意味着什么？这意味着，数组本身就是有序的，因为排序前后的数组相同。只要b和e不重合，就代表有不同的地方，在计算长度的时候要+ 1（比如计算日期，周一到周五有几天？5 - 1 + 1 = 5天）。这里利用了一下true会被转换成1，false会被转换成0的性质。

# 单调栈

排序的方法还可以再优化。仔细想想，排序的目的是什么？找到原数组和排序数组的不同。从另一个角度看，找到原数组和排序数组相同的地方，也可以达到同样的目的。而题目里是升序，这就可以用到单调栈了：

function findUnsortedSubarray(nums: number[]): number {
  const length = nums.length;
  let stack = [];

  let b = length - 1,
    e = 0;
  for (let i = 0; i < length; ++i) {
    while (stack.length && nums[stack[stack.length - 1]] > nums[i]) {
      b = Math.min(b, stack.pop()!);
    }
    stack.push(i);
  }
  stack = [];
  for (let i = length - 1; i >= 0; --i) {
    while (stack.length && nums[stack[stack.length - 1]] < nums[i]) {
      e = Math.max(e, stack.pop()!);
    }
    stack.push(i);
  }

  return Math.max(e - b + 1, 0);
}

从前往后是一个递增栈，而从后往前是一个递减栈，因为从后往前是递减的。
