# [LeetCode 671] 迭代遍历 + 排序

用的是中序遍历，然后在遍历的过程中保存每个结点的值，然后从小到大排序，取出第二个不重复的值。

function findSecondMinimumValue(root: TreeNode | null): number {
  const res: number[] = [];
  const stack: TreeNode[] = [];
  while (root || stack.length) {
    while (root) {
      stack.push(root);
      root = root.left;
    }
    root = stack.pop()!;
    res.push(root.val);
    root = root.right;
  }
  res.sort();
  let length = res.length;
  for (let i = 1; i < length; ++i) {
    if (res[i] !== res[i - 1]) return res[i];
  }
  return -1;
}

为什么是中序遍历？一开始我以为这个和二叉搜索树有相同的性质，后来发现不是；但不想改了。
