# [LeetCode 297] 每天一个被拒小技巧——JSON

直接用JSON操作来进行序列化和反序列化。

这种方式对"JSON安全"的对象来说应该是最好的方式之一了。

var serialize = function (root) {
  return JSON.stringify(root);
};

var deserialize = function (data) {
  return JSON.parse(data);
};
