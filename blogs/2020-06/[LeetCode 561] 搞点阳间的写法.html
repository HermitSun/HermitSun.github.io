# [LeetCode 561] 搞点阳间的写法

没错这次仍然是函数式编程。

需要注意的是，如果sort不自定义排序函数，会先转换成字符串，然后按照字典序进行排序。

function arrayPairSum(nums: number[]): number {
  return nums
    .sort((a, b) => a - b)
    .filter((_, i) => i % 2 === 0)
    .reduce((a, b) => a + b);
}
