# [LeetCode 10] 每天一个被拒小技巧——eval

没错，就是那个臭名昭著的eval，但是可以解决很多问题。

当然最好不要在实际开发中使用，因为无论是安全性还是性能，都有可能存在问题。

function isMatch(s: string, p: string): boolean {
  return eval(`/${p}/.test(${s})`);
}

当然也可以用RegExp对象，但经过实际运行发现，eval要更快一点。

function isMatch(s: string, p: string): boolean {
  return new RegExp("^" + p + "$").test(s);
}
