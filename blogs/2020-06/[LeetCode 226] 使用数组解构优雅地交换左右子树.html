# [LeetCode 226] 使用数组解构优雅地交换左右子树

思路很简单，就是递归交换左右子树。数组解构可以很优雅地完成交换的任务。

function invertTree(root: TreeNode | null): TreeNode | null {
  function swap(root: TreeNode | null) {
    if (root) {
      [root.left, root.right] = [root.right, root.left];
      swap(root.left);
      swap(root.right);
    }
  }
  swap(root);
  return root;
}
